css属性:背景与边框

发表于 2018-08-02 | 分类于 css basic

边框

border

定义元素边框的外观特性

border-width

定义元素的边框厚度

border-style

定义元素的边框样式

border-color

定义元素的边框颜色

border-shadow

定义元素的阴影

结合伪类实现阴影

border-radius

定义元素的阴影

border-image

定义将图像应用到元素的边框上

背景

background

定义元素的背景特性

  • 一个元素可以设置多组背景图像，每组属性间使用逗号分隔。（背景色background-color不能设置多组，背景色通常都定义在最后一组上）
  • 如果设置的多组背景图之间存在着交集（即存在着重叠关系），前面的背景图会覆盖在后面的背景图之上。

background-color

定义元素使用的背景颜色

  • 背景图像会覆盖在背景颜色。

background-image

定义元素使用的背景图像

linear-gradient

  • 渐变图标 http://lea.verou.me/css3patterns/

background-repeat

定义元素的背景图像如何填充。

background-attachment

定义滚动时背景图像相对于谁固定

background-position

指定背景图像在元素中出现的位置

  • 雪碧图 https://www.toptal.com/developers/css/sprite-generator
  • .icon2{background-position: -40px -40;}

background-origin

指定的背景图像计算background-position时的参考原点(位置)

background-clip

指定对象的背景图像向外裁剪的区域

  • 文字遮罩 http://demo.doyoe.com/css3/background-clip/mask-text2.htm

    background-size

    定义背景图像的尺寸大小
